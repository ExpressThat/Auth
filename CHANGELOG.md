# Changelog

All notable changes to ExpressThat Auth are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and released public
surfaces follow [Semantic Versioning](https://semver.org/).

## Unreleased

### Added

- Established the TypeScript Turborepo foundation, strict quality gates,
  dual-runtime contract spikes, security scanning, and safe repository
  scaffolding.

### Changed

- Documentation is now written and validated alongside every implementation
  task rather than deferred until release.

### Internal

- Defined independent public-surface versions bound by an immutable platform
  release manifest, task-time change fragments, migration notes, and
  coordinated security advisories.

No general-availability release has been published.
