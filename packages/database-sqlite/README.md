# SQLite Database Adapter

Status: planned package shell.

Owns the Drizzle SQLite schema, migrations, and repository implementation. It
implements `@expressthat-auth/data-access` and passes the shared database
conformance suite.

SQLite is the initial local-development default and a supported self-hosted
choice. See [ADR-0022](../../docs/decisions/0022-pluggable-database-adapters.md).
