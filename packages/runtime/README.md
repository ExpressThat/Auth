# Runtime Contracts

Status: planned package shell.

Owns runtime-neutral ports for time, randomness, cryptography, queues, caches,
objects, secrets, keys, telemetry, email, identity providers, and other
replaceable capabilities.

Contracts declare capabilities, failure semantics, tenant scope, runtime
support, and security properties without importing provider SDKs. See the
[workspace register](../../docs/architecture/workspace-ownership.md).
