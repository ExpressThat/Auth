# Cloudflare Deployment

Status: planned deployment shell.

Owns Workers entry points, bindings, configuration, and hosted composition. It
selects adapters and composes applications without moving Cloudflare types into
runtime-neutral product packages.

The hosted service may publish contractual residency, reliability, automation,
and support commitments. Those commitments do not apply to self-hosted
deployments. Docker remains a permanent behaviorally equivalent target. See the
[responsibility boundary](../../docs/operations/hosted-self-hosted-responsibility.md).
