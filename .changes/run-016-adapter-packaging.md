Define and enforce independently packaged infrastructure adapters.

- Reserve singular provider-package prefixes for queue, cache, object storage,
  secrets, key management, observability, DNS, certificates, and deployments.
- Require explicit public exports and a production runtime-contract dependency.
- Validate statically readable Node, operating-system, Docker-architecture, and
  external-capability support metadata.
- Record the security, portability, hosted, and self-hosted consequences in
  ADR-0025 and the architecture guide.
